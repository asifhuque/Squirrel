Maven - FourSquare C# API 0.0.9
===============================

Easy to use asychronous API useful for building Silverlight / Windows phone foursquare apps.


Requirements:
Visual 2010 express.
Windows phone SDK for running the windows phone 7 solution.


Gettting started:

Update the app.config or the constant class (for windows phone 7) with your foursquare user/pass to run the tests. 



Currently that's it. Enjoy  + feel free to contribute.
